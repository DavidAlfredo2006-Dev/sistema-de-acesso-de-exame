Sobre o Sistema

O Sistema de Exame de Acesso é uma plataforma online restrita ao Instituto Politécnico Industrial Alda-Lara (IPIAL), projetada para gerenciar o processo de inscrição e seleção de candidatos para vagas de estudo.

Funcionalidades
(Fase 1 - Interface do Sistema)

  • Inscrição online para candidatos
  • Gerenciamento de dados de candidatos (somente visualização)
  • Acesso restrito a administradores do IPIAL Tecnologias Utilizadas (Fase 1)
  • HTML5 para estruturação do conteúdo
  • CSS3 para estilização e layout
  • JavaScript para interatividade e dinamismo

Fases Futuras

  • Fase 2: Desenvolvimento do Back-End com NodeJs e/ou PHP
  • Fase 3: Integração com banco de dados MySQL para armazenamento de dados do usuário
  • Fase 4: Implementação de funcionalidades de seleção e classificação de candidatos Objetivos
  • Simplificar o processo de inscrição e seleção para o IPIAL
  • Reduzir o tempo de processamento de inscrições
  • Aumentar a eficiência e a transparência no processo de seleção

Público-Alvo

  • Estudantes que desejam se inscrever no IPIAL
  • Administradores do IPIAL responsáveis pelo processo de inscrição e seleção
